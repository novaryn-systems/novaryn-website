Security Measures

Layered Security by Design

AxJedi uses multiple defensive controls across network, transport, application, and operational layers.

Network and Edge Controls

  • Cisco ASA firewall with source IP allowlist rules.
  • Caddy reverse proxy enforcing TLS 1.3 with Let's Encrypt certificates.
  • Application-level IP allowlist checks to enforce approved access paths.

Authentication and Session Security

  • Flask-Login for authenticated session handling.
  • bcrypt password hashing for credential storage.
  • Secure cookie configuration and CSRF token protection.
  • Login rate limiting to reduce brute-force risk.

Monitoring and Abuse Controls

  • Audit logging for access and key application actions.
  • Daily query limits to reduce misuse and improve platform stability.
